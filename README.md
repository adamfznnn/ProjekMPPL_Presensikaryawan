# Sistem Presensi Karyawan Berbasis Web  
## (PHP & MySQL)

Aplikasi **Sistem Presensi Karyawan** berbasis web yang dikembangkan menggunakan PHP Native** dan **MySQL**.  
Aplikasi ini dibuat sebagai **tugas kelompok** pada mata kuliah **Manajemen Proyek Perangkat Lunak**.

Sistem ini dirancang untuk membantu proses pencatatan kehadiran karyawan secara **efektif, terstruktur, dan terkomputerisasi**, serta mendukung pengelolaan data karyawan, presensi harian, pengajuan izin, dan pengelolaan lokasi kerja dengan pembagian hak akses **Admin** dan **Karyawan**.
